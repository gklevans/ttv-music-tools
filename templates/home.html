Welcome!

These tools provide information obtained from StreamerSonglist.

Request History Lookup

This tool lets you look up the history of requests made through StreamerSonglist in any streamer's channel under a given username.
Useful if you'd like to see what song requests you've made before in a stream.
You may also include the titles of scores from any MuseScore links placed in a queue as a live-learn.
Note that any different names used (e.g. previous usernames or manually entered names) must be searched separately.

Songlist Settings Lookup

This tool lets you look up the songlist settings of any streamer who uses StreamerSonglist.
Useful if you'd like to check whether and how often you may request songs in a stream.
Note that these settings may not exactly reflect the request procedures in place for each stream.
Live-learn settings are not included in these results.
